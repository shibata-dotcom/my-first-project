# 柴田哲也 税理士 自己紹介ページ

税理士・柴田哲也（哲也事務所）の自己紹介ページです。

## 概要

シンプルで見やすいデザインの自己紹介サイトです。

## 主な機能・デザイン

- プロフィール情報の表示（氏名・職業・事務所）
- ご挨拶・メッセージセクション
- キャラクターアイコン（DiceBear によるイラスト風アバター）
- 薄い青を基調としたグラデーション背景
- 左右に自然の風景画像を配置
- スマートフォン対応（レスポンシブデザイン）

## 使用技術

- HTML5 / CSS3
- [DiceBear](https://dicebear.com/)（アバター生成 API）
- [Unsplash](https://unsplash.com/)（自然の風景写真）
